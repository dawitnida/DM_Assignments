Data linking system . There is disclosed a data linking system which efficiently links data for improved performance and quality of the system. A data storage device stores data. A transmitter-side storage information management device manages storage information concerning control of storage of the data in the data storage device. A transmitter-side storage information interface device transmits and receives the storage information for linkage with a receiver side. A data writing device writes the data into the data storage device based on the storage information. A receiver-side storage information management device manages the storage information concerning control of storage of the data in the data storage device. A receiver-side storage information interface device transmits and receives the storage information for linkage with the transmitter side. A data reading device reads the data from the data storage device based on the storage information.
