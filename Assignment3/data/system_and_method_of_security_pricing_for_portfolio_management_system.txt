SYSTEM AND METHOD OF SECURITY PRICING FOR PORTFOLIO MANAGEMENT SYSTEM . A portfolio pricing system and method can include a database receiving multiple feeds, an analytics server in communication with the database, and an accounting server. The analytics server can include a pricing algorithm that assigns prices to the analytics server and the accounting server according to a hierarchy that selects the most reliable source for each type of security. The reliability may be determined within a date range or according to the source of the price.
