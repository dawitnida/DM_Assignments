Assigning tangible assets to workplaces . A method for assigning tangible assets to workplaces includes receiving input data provided at an input interface, the input data including information about a situation that requires to update a structure of current assignments of specific tangible assets to specific workplaces, selecting, dependent upon the information of the received input data, a workplace profile from a profile database, wherein the workplace profile corresponds to a workplace type and includes a predefined set of asset types, of which types specific tangible assets can be to be assigned to a specific workplace of the workplace type, retrieving, from an assignment database, assignment data including information about the current assignment structure, comparing the selected workplace profile with the retrieved assignment data, determining an update proposal for updating the current assignment structure, and providing output data to an output interface, the output data including information about the update proposal.
