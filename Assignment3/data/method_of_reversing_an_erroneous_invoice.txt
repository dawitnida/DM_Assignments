Method of reversing an erroneous invoice . A method of reversing a plurality of erroneous invoices proffered by a plurality of vendors/suppliers. The method comprises providing a master database for inputting a plurality of vendor invoices, storing a cancellation request for each of the plurality of erroneous invoices in a response database, executing an agent to extract data pertaining to the cancellation request stored for each of the plurality of erroneous invoices from the response database, creating an intermediate file containing data extracted from the response database, executing a delivery component for adding a header record and trailer record to the intermediate file, and for renaming the intermediate file, and executing a bridge program for delivering the intermediate file from the response database into the master database and processing the intermediate file in the master database in order to reverse each of the plurality of erroneous invoices in the master database.
