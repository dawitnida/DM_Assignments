System and method for online valuation and analysis . The invention provides an online valuation system and method, which provides an user with analytic tools used for valuing a company. The system includes a web server providing online access to a user performing valuation, an internal financial database which retrieves financial data and assumptions from external databases, an analytical tool which retrieves the data from the internal financial database and process the data based on the assumptions. The user may choose an analytic tool suitable for his or her valuation, and the system provides the user with a real time answer. The system updates the financial data and assumptions it keeps at least once a day.
