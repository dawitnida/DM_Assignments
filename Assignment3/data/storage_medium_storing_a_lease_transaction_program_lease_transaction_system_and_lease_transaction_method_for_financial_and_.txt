Storage medium storing a lease transaction program, lease transaction system and lease transaction method for financial and related instruments . The present invention is a computer-readable storage medium storing a lease transaction program for financial and/or related instruments, a lease transaction system for financial and/or related instruments, and a lease transaction method for financial and/or related instruments, in which a lease transaction market for financial instruments and/or quasi-financial instruments is created via at least one computer network and users can conclude lease transactions between and/or among themselves.
