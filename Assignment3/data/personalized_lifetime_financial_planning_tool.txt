Personalized lifetime financial planning tool . A networked computer system comprising a user, a website of a bank or a financial institution, and a computer software tool. The computer software tool is accessible on the website of the bank or the financial institution and provides personalized lifetime financial planning to the user based upon financial information of the user along with a life event and a life stage of the user in an interactive computer generated timeline. The timeline comprises a past event, a present event, a future event, or a combination thereof of the user. The tool provides an automated reminder to the user based upon information of the user present in the timeline. The user may be a customer of the bank or the financial institution.
