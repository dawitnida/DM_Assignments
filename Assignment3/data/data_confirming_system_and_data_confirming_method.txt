Data confirming system and data confirming method . What is disclosed is a data confirming system comprising: an image reader for reading an image of a manuscript written in a financial sheet, a character reader for transforming said image to a text data corresponding to said image data, a data dividing section for dividing said image and said text data, into each item in said financial sheet, a searching data obtaining section for obtaining a searching data from said text data divided into said item, a searching data confirming section for confirming said searching data, a database for storing each financial trade data made in each past financial trade, a data searching section for searching said database for a financial trade data corresponding to said searching data, and a data distributing section for distributing each item of said financial trade data to each of confirming terminals, after extracting each item of said financial trade data from said financial trade data.
