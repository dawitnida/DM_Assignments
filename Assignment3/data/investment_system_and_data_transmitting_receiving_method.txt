Investment system and data transmitting/receiving method . A server device (10) stores at least investment target data made up of music tune data and data related to an artist. An investment client device (20) accesses the server device (10) to download the investment target dat and transmits investment data for an artist or a music tune which a user thinks to become popular in the future, to the server device (10). Thus, the user can invest in a new music tune or artist by using a network such as the Internet.
