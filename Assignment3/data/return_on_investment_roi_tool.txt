Return on investment (ROI) tool . The present invention provides a system and related method that enables a user to use one or more appropriate ROI tools according to the needs of the users. Specifically, the user provides data related to the organizations and technical expenditures, and this information is used to determine an applicable set of ROI tools. The inputted data is then provided to the applicable ROI tools for automatically examining a company's financial data, and using this evaluation to assess the desirability of technology expenditures.
