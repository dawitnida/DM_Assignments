Refinancing Program for an Underwater Loan . A refinancing program for an underwater loan is disclosed. In particular embodiments, a method for refinancing a loan includes determining an original loan value, wherein the original loan value is associated with an original loan secured by a property having a current property value that is less than the original loan value. The method also includes calculating, with a processor, a conforming loan value based, at least in part, on the current property value, wherein the conforming loan value is associated with a conforming loan and the conforming loan value is less than the current property value. The method also includes calculating a shared appreciation mortgage (SAM) loan value based on a numerical difference between the conforming loan value and the original loan value.
