Virtual reality generator for use with financial information . A virtual reality generator having an input module that receives as input financial information is disclosed. The virtual reality generator outputs to a display device a virtual reality world generated from the financial information. The financial information can be pre-processed by a financial analytic system prior to input to the virtual reality generator. The financial information can be received from a data file. The virtual reality generator can dynamically display and continuously update the virtual reality world. Further, movement through the virtual reality world can be simulated.
