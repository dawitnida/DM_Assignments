Method and system of data analysis for the detection of fraudulent financial transactions . A system and method for detecting and analyzing information units and particularly the analysis and evaluation of data records by selecting particular aspects of the data record is disclosed. The method comprises creating a complexity catalog based on an information unit and establishing at least one score unit based on the at least one complexity catalog. The system comprises an infrastructure server device to create at least one complexity catalog, and a complexity catalog to hold at least one list of ordered complexity values associated with partitioned sub-unit blocks, an application server to build at least one information summary unit based on an information unit and on an associated complexity catalog, and a scoring component to provide scores.
