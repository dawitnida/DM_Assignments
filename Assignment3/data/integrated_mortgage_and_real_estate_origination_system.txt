INTEGRATED MORTGAGE AND REAL ESTATE ORIGINATION SYSTEM . Computer software for generating customer leads comprises a data input module for receiving potential customer data from an outside source and converting the data into a format compatible with preexisting software of an end user, a data storage module for storing the converted data, and a processing module for processing the converted data to determine a transaction type and for forwarding the converted data to the user based upon the determined transaction type. The converted data is displayable on a computer screen of the user by the user's preexisting software and provides the determined transaction type and client contact information.
