System and method for determining a lifecycle stage of a corporation using quantitative measures . A system for determining the lifecycle stage of a corporation using quantitative measures is provided. The system comprises a lifecycle model definition component. The lifecycle model definition component is configured to define one or more lifecycle models. Each lifecycle model is associated with a set of quantitative measures and a set of rules. The system further comprises a lifecycle stage inference component and a lifecycle stage determination component. The lifecycle stage inference component is configured to infer a particular lifecycle stage for the corporation based on the set of quantitative measures and the set of rules associated with each lifecycle model. The lifecycle stage determination component is configured to combine the inferences made by the lifecycle inference component to determine an overall lifecycle stage for the corporation.
