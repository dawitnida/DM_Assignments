System and method for resolving transactions using weighted scoring techniques . A system and method for the online settling of a transaction is provided. The system includes a server having a rules configured to process available data regarding the debtor party using rules established on behalf of the party to the financial transaction and weighting factors accorded to selected available data related to the debtor party. The rules and weighting factors are employed by the rules engine to establish a transaction settlement offer set conveyable to the user and comprising multiple simultaneously presentable and individually selectable offers, each offer configured to settle the transaction on terms satisfactory to the party to the transaction.
