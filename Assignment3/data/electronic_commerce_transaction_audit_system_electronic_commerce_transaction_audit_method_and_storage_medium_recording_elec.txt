ELECTRONIC COMMERCE TRANSACTION AUDIT SYSTEM, ELECTRONIC COMMERCE TRANSACTION AUDIT METHOD, AND STORAGE MEDIUM RECORDING ELECTRONIC COMMERCE TRANSACTION AUDIT PROGRAM THEREON . The present invention comprises agent for gathering transaction log means for automatically collecting all exchange messages notarized and recorded depressively by a plurality of electronic notarize means in connection with the electronic commerce transaction so as to reproduce them as an event of the entire wide network area, agent for gathering protocol standards means for automatically collecting protocols of the specifications on the electronic commerce transaction, whereby correctly grasping an event to be generated in the entire wide network area, and a log analysis engine for comparing the event of the entire wide network area reproduced by the agent for gathering transaction log means with the event to be generated in the entire wide network area grasped by the agent for gathering protocol standards means, whereby carrying out an objective audit.
