AUTOMATIC FINANCIAL INSTRUMENT TRANSACTION SYSTEM . A computer-based transaction system manages representations of a plurality of positions in a first type of financial instrument, such as bond future contracts. The transaction system, at a first predetermined time, converts each position in the first type of financial instrument into a corresponding position in a second type of financial instrument, such as bonds. At a second predetermined time that is after the first predetermined time, the transaction system converts each position in the second type of financial instrument into a position in the first type of financial instrument.
