Media wall for displaying financial information . A system dynamically displays graphic identifier information, such as corporate logos, and value information for financial instruments, such as recent trade information, on a video wall having several monitors arranged to form a larger display. The system has input ports to receive feeds with identifiers and corresponding values of financial instruments, a filter to extract the identifiers and corresponding values of the financial instruments, a memory to store the extracted information and a table associating the financial instrument identifiers with graphic symbols, and processors to form a display signal with the graphic symbols and values.
