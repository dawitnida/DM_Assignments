Monitoring a status of a database by placing a false identifier in the database . Embodiments include a system, an apparatus, a device, and a method. An embodiment provides an apparatus that includes a manager module operable to plant a fake financial transaction card identifier in a customer database that includes at least one genuine financial transaction card identifier. The apparatus also includes a control module operable to maintain the fake financial transaction card identifier in a hidden state. The apparatus further includes a security assessment module operable to periodically monitor a usage state of the fake financial transaction card identifier.
