System and method for commoditizing browsing time in a self-service store . A system and method commoditize browsing time in a self-service store for a communication carrier. The method comprises communicating with a user, enabling the user to browse the store, tracking the amount of time the user spends browsing the store, calculating a charge, if any, for the amount of time the user spent browsing the store, and, if a charge is incurred, charging the user's account for such charge. In one embodiment, the self-service store is a voice store in which a user calls into the store. However, the invention is not limited to voice stores, and the invention could apply to stores accessed via text messaging, the web, etc. Communication with a user can be initiated by the user or by the self-service system.
