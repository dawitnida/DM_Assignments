METHOD FOR PLACING ADVERTISEMENT ON WEB PAGES . A method for placing advertisements on a web page is disclosed which comprises providing a web page having a designated space for advertisement, providing a clickable member on the web page with a link to a procurement process, once the clickable member being clicked, the procurement process is invoked, uploading an advertisement upon a completion of the procurement process, placing the uploaded advertisement solely on the designated space of the web page, and displaying the web page with the uploaded advertisement when the web page is requested.
