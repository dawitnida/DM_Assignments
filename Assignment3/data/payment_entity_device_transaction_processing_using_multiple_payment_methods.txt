PAYMENT ENTITY DEVICE TRANSACTION PROCESSING USING MULTIPLE PAYMENT METHODS . A method begins with a payment entity device receiving an accounts payable data file from a client entity processing module. The method continues with the payment entity device determining a method of payment for an account payable of the accounts payable data file in accordance with a payables profile. The method continues with the payment entity device initiating a first type of payment for the account payable when the method of payment is a first type; initiating a second type of payment for the account payable, when the method of payment is a second type, or initiating a third type of payment for the account payable when the method of payment is a third type.
