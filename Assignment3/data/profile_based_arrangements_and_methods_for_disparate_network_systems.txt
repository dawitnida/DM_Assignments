Profile-Based Arrangements and Methods for Disparate Network Systems . Electronic transaction data sets are processed for a multitude of disparate transactions using a plurality of autonomous payment networks. A software-programmed computer circuit receives and processes electronic transaction data to identify and use electronic ID data to retrieve stored profile data. The computer circuit then uses the retrieved profile data to select one of a plurality of payment networks available to each of a buyer and seller involved in the transaction. Electronic payment is carried out using the selected payment networks.
