System and method for managing bad accounts of accounts receivable . A system and method for calculating and balancing bad accounts of accounts receivable according to types and ages of the accounts receivable. The system includes a database server (2) for storing accounts receivable data; an application server (3) electrically connected with the database server for accessing and processing data stored in the database server; and a plurality of client computers (1) electrically connected to the application server for downloading data from and uploading data to the database server. The application server includes an accounts receivable managing module (30) for managing and updating the accounts receivable data stored in the database server, an account age analyzing module (32) for analyzing ages of the accounts receivable, a bad account provision calculating module (33) for calculating bad account provisions of the accounts receivable.
