GENERATING AN ALERT BASED ON ABSENCE OF A GIVEN PERSON IN A TRANSACTION . Techniques for generating an alert based on absence of a given person in a transaction are provided. The techniques include monitoring, via video, a transaction, wherein the transaction includes presence of a given person in the transaction, relating the video of the transaction to a corresponding portion of a transaction log (TLOG), using the video and corresponding portion of the TLOG to detect if the given person in the transaction is present, and generating an alert if the given person is not present at the transaction.
