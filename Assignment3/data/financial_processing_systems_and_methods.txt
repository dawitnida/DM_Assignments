Financial processing systems and methods . A database system that can analyze multichannel data multilaterally so as to total data items and satisfy various detailed conditions. The system includes: a data storage unit 50 for storing data; a data base management unit for managing input/output of data to/from this data storage unit; a data processor for controlling the data base management unit to obtain predetermined data from the data storage unit and process the data; and a stored function 40 enabled to define a field value used to group data. The data processor is used to enter an execution command that specifies the stored function to be used in a data processing so as to instruct the data base management unit to total data items according to the field value defined by the stored function specified in the execution command.
