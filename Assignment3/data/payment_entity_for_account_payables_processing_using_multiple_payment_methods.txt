PAYMENT ENTITY FOR ACCOUNT PAYABLES PROCESSING USING MULTIPLE PAYMENT METHODS . A method begins by receiving an accounts payable data file from a client device. The method continues by determining whether a payables profile of a client associated with the client device is to be modified based on the accounts payable data file. The method continues by determining a level of service of the client when the payables profile is not to be modified. The method continues processing payment transactions for accounts payable contained in the accounts payable data file on behalf of the client in accordance with the payables profile via a wide area network when the level of service is a first level of service.
