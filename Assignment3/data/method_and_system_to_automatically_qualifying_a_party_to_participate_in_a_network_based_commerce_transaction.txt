Method and system to automatically qualifying a party to participate in a network-based commerce transaction . A system to facilitate network-based commerce includes receiving information relating to an item to be transacted via a network-based commerce system from a first party. Criterion information specifying at least one criterion to be satisfied by a second party in order for the second party to be qualified to transact for the item via the network-based commerce system is received from the first party. An automatic determination is made as to whether the second party satisfies the at least one criterion and if so, then the second party is automatically qualified to transact for the item via the network-based commerce system.
