SYSTEM FOR COST-SENSITIVE AUTONOMOUS INFORMATION RETRIEVAL AND EXTRACTION . A multi-agent system is provided for automatically acquiring desired information from one or more information sources. The multi-agent system includes a plurality of data provider filter agents associated with the one or more information sources. The data provider filter agents are configured to search for the desired information within the respective information sources based on an assessment of the one or more information sources. The multi-agent system also includes a content extraction agent configured to acquire a plurality of articles containing the desired information from the one or more information sources based on the search.
