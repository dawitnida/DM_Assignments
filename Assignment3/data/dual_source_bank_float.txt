DUAL SOURCE BANK FLOAT . Bank float is assigned using a dual source bank float system by first determining which of a plurality of checks will be processed via a paper route and which will be processed via an image route. For the checks that will be processed via the paper route, a set of rules will be determined. For the checks that will be processed via the image route, a different set of rules will be determined. A bank float is applied to the paper checks using the paper set of rules and bank float is assigned to the image checks using the image set of rules.
