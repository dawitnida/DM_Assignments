Wireless networking cash register . A wireless networking cash register comprises a cash register, a wireless networking HUB, a code generator and a charge counter; the cash register has therein an electric circuit board electrically connecting with the code generator and the charge counter, and has externally thereof a display, a printer and an inputting keyboard. The wireless networking HUB is built in the cash register, and has a set of antenna, at least a wide area network (WAN) interface and a local area network (LAN) interface. Thereby, the code generator in the cash register generates a code for networking of the user, and the charge counter starts to calculate the charge for networking. The display displays the trading information, and the printer prints the detail statement of trading. The cash register is applicable to the business activity fields such as general cafs, network cafs, breakfast shops, restaurants and hotels etc.
