System, method and computer program product for habitat-based universal application of functions to network data . A system, method and article of manufacture are provided for applying a function to a habitat for universal application of a function to data for output on a remote client device. Data from at least one network-based data source is retrieved utilizing user-defined information. The retrieved data is aggregated in a habitat located remotely from the user. A function is sent to the habitat. The function is applied to the aggregated data in the habitat. The data is transmitted to the client device.
