Depositing items of value . A method of depositing items of value, the method comprising 1) receiving (30) a tamperevident and coded container holding the items of value, the code relating to the content of the container; 2) verifying (32) that the package has been received in an acceptable manner, and, if it has (36), 3) either incrementing (38) an account with the value of the items of value in the container by reference to the code and without opening the container, or, if the account has previously been incremented, leaving the account unchanged.
