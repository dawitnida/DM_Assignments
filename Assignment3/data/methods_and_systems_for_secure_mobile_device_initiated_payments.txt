METHODS AND SYSTEMS FOR SECURE MOBILE DEVICE INITIATED PAYMENTS . "Systems, methods, processes, computer program code and means for conducting a transaction include receiving a request to authorize a purchase transaction, the request including a static virtual payment account number (""VPAN""), an expiry date, and a dynamic code generated by a mobile device, the request further including a transaction amount and a transaction date, identifying, based on the static VPAN, a payment card account number, verifying that the dynamic code matches an expected value of the dynamic code, and transmitting an updated authorization request message to an issuer associated with the payment card account number for authorization analysis."
