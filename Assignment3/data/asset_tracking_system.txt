ASSET TRACKING SYSTEM . An asset tracking system for tracking assets, for example, a vehicle, with an integrated topology that is configured minimize the complexity of the system by eliminating hardware duplication and sensing the ignition state while obviating the need to access the ignition switch circuit; the asset tracking system optionally including a loan obligation management system that is integrated with the asset recovery system which provides in-vehicle notifications of loan obligations for the asset upon which the loan obligation is based from a remote location directly to the asset.
