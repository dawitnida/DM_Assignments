CONSUMER TRANSACTION FACILITY . In a consumer transaction facility connected via a communication line 14 to a host computer, a display and input control unit performs on a display device a screen display in which a destination electronic mail address for transaction records is able to be inputted. When the destination electronic mail address for transaction records is inputted through an input unit, the display and input control unit performs a screen display in which it is selectable as to whether the transaction records are to be transmitted, and in addition performs a screen display in which it is selectable as to whether the transaction records are to be issued in the form of a sheet receipt.
