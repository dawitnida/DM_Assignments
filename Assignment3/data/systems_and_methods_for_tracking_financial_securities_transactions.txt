Systems and methods for tracking financial securities transactions . Systems and methods for tracking financial securities transactions are provided. More specifically, price histories for one or more financial securities are graphically represented using a visual display interconnected with a computer. Indicators are provided on the graphical representation of the one or more securities indicating an action done to the financial security or securities at a point in time. By activating an indicator, information is communicated to the user relating to the action done to the financial security or securities.
