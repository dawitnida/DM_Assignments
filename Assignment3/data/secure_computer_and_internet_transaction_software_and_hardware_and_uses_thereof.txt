Secure computer and internet transaction software and hardware and uses thereof . Methods of using an alternate embedded browser object in conjunction with an ecommerce transaction software system are described that comprise providing a server; providing a client; providing a user; and utilizing a software application, a code, a password or a combination thereof for server and client authentication, wherein the software application, a code, a password or a combination thereof is based on the transaction type requested by the user at the time of use. Also described is an embedded browser object that interfaces to an ecommerce transaction system without the need for a keyboard.
