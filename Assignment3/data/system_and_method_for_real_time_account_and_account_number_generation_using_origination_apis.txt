SYSTEM AND METHOD FOR REAL TIME ACCOUNT AND ACCOUNT NUMBER GENERATION USING ORIGINATION APIS . A system and method generate an account in real time in accordance with an application programming interface (API). The API contains parameter descriptions listing universal resource locator (URL) parameters associated with items. A format for implementing an http request to transmit data to in compliance with the defined format is disclosed. A transparent mode for transmitting a response to an http request transmitting data provides for the transmission of an extensible markup language (XML) formatted file communicating an outcome to the request.
