Pattern matcher . A user can select from one of a number of predefined patterns, create a new pattern, or import a pattern from existing data. With a time interval specified, a search is performed and comparable patterns identified in one or more streams of securities data. Filtering can be performed before and/or after the searching with, for example, a number of the closest pattern matches displayed to the user. These closest pattern matches can be displayed graphically, in a table, in a list, or in general in any manner which may be useful for analysis by the user. One or more of the system and the user can also place a trade (buy or sell) of one or more of the identified closest matches.
